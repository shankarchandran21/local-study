# What is Overriding ?
Overriding means writing a new version of a method (function) in the child class that already exists in the parent class.

# What is Method Overloading ?

In python overloading will not work because python will not support this

Method Overloading (General Meaning)
Method overloading means using the same function/method name in a class but with different parameters.

👉 Same method name
👉 Different number of parameters or different types of parameters

```
add(5, 10)        → adds two numbers  
add(5, 10, 15)    → adds three numbers  
add("Hi", "😄")   → joins two strings

```
