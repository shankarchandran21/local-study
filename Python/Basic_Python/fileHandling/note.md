Mode  | Description
      |
'r'   | Read-only (file must exist)
'w'   | Write-only (overWrites or creates)
'a'   | Append-only (adds to end of file)
'r+'  | Read + Write (file must exist)
'w+'  | Write + Read (overwrites or creates)
'a+'  | Append + read (creates if not exists)
'rb'  | Read Binary
'wb'  | write Binary
'ab'  | Append Binary
