# Diff of Automation and scheduling 

| Feature        | Automation                                        | Scheduling                        |
| -------------- | ------------------------------------------------- | --------------------------------- |
| **Purpose**    | Reduce manual work by automating tasks            | Define **when** a task should run |
| **Trigger**    | Can be event-driven or time-driven                | Usually time-driven               |
| **Complexity** | Can involve multiple steps, workflows, conditions | Usually single task at a set time |
| **Dependency** | May or may not require schedule                   | Depends on schedule to run        |


# Python Architecture:

Source Code -> Compiler -> Intermediate Code (Byte Code) -> PVM (Python virtual Machine) -> Machine Code -> Execution

"Source Code" to "Intermediate Code" is CompileTime
"PVM" to "Machine Code" is Runtime
